## 🚀 Quick Start

### Prerequisites

- **Python 3.11+** 🐍
- **pip** (Python package manager)
- **Git**
- **Modern web browser** (Chrome, Firefox, Safari, Edge)

### ⚡ One-Command Setup

```bash
# Clone the repository
git clone https://github.com/batmandevx/mifid.git

# Navigate to project
cd mifid/regulatory-risk-system

# Run the application (automatic setup)
./start.sh
```

**That's it!** The script automatically:
- ✅ Creates Python virtual environment
- ✅ Installs all dependencies
- ✅ Sets up database
- ✅ Starts backend API (port 8000)
- ✅ Starts frontend (port 8080)

### Risk Calculation Methods

#### Value at Risk (VaR)

**Mathematical Foundation:**

```
VaR_α = inf{l ∈ ℝ: P(L > l) ≤ 1 - α}
```

**Implementation Methods:**

| Method | Formula | Best For |
|--------|---------|----------|
| **Historical** | `quantile(returns, 1-α)` | Non-normal distributions |
| **Parametric** | `μ - z_α × σ` | Normal distributions |
| **Monte Carlo** | `percentile(simulations, 1-α)` | Complex derivatives |
| **Cornish-Fisher** | `μ - z̃_α × σ` | Skewed distributions |

#### Expected Shortfall (ES)

```
ES_α = E[L | L > VaR_α] = 1/(1-α) ∫_α^1 VaR_u du
```

#### GARCH(1,1) Volatility

```
σ²_t = ω + α × r²_{t-1} + β × σ²_{t-1}
```

Where:
- ω = long-term average variance
- α = reaction to shocks
- β = persistence (α + β < 1 for stationarity)

## 🔌 API Reference

### Authentication

```bash
# Login and get JWT token
curl -X POST "http://localhost:8000/api/auth/login" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "admin",
    "password": "password"
  }'

# Response
{
  "access_token": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9...",
  "token_type": "bearer",
  "expires_in": 1800
}
```

### Core Endpoints

#### Datasets

```bash
# Upload dataset
POST /api/datasets/upload
Content-Type: multipart/form-data
file: <CSV or Excel file>

# List datasets
GET /api/datasets?page=1&limit=20

# Get dataset details
GET /api/datasets/{dataset_id}
```

#### Analysis

```bash
# Run risk analysis
POST /api/analyses
{
  "dataset_id": "uuid",
  "name": "Q1 Risk Assessment",
  "analysis_type": "full",
  "regime": "basel_iii",
  "confidence_level": 0.99,
  "time_horizon": 10,
  "var_method": "historical"
}

# Get analysis results
GET /api/analyses/{analysis_id}

# List analyses
GET /api/analyses?status=completed&page=1
```

#### Reports

```bash
# Generate report
POST /api/reports
{
  "analysis_id": "uuid",
  "report_type": "excel",
  "regime": "basel_iii"
}

# Download report
GET /api/reports/{report_id}/download
```

### WebSocket Real-time Updates

```javascript
const ws = new WebSocket('ws://localhost:8000/ws');

ws.onmessage = (event) => {
  const data = JSON.parse(event.data);
  console.log('Analysis progress:', data.progress);
};

// Subscribe to analysis updates
ws.send(JSON.stringify({
  type: 'subscribe',
  analysis_id: 'uuid'
}));
```
